ArcOS

A super advanced Electron OSE


ArcOS has been in development since February 17th, and has improved tremendously since. It is based on Electron, without any frameworks, just vanilla JS. This project really proves how much is possible without frameworks.

It sports flexible application loader, file management with actual access to all storage on your computer, modular theming, separate user accounts, and a whole ton more features for which there is not enough space on this page for.
